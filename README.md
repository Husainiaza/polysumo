| Input     | Nombor GPIO Pin     | Aktuator/Sensor      |
| :-------- | :-------            | :------------------- |
| Motor A1  |  `17`               | **Motor A DC Kiri**  |
| Motor A2  |  `12`               | **Motor A DC Kiri**  |
| Motor B1  |  `18`               | **Motor B DC Kanan** |
| Motor B2  |  `19`               | **Motor A DC Kanan** |
| Buzzer    |  `03`               | **Buzzer(bunyi)**    |
| Trigger   |  `2`                | **Ultrasonic**       |
| Echo      |  `4`                | **Ultrasonic**       |
